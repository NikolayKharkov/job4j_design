Создан класс User. Данный класс содержит следующие поля : String name, int age, 
boolean active. Переопрелен метод finalize.
Теоретический расчет:
заголовок объекта занимает 12 байт
0 + 12 = 12
тип int занимает 4 байта
12 + 4 = 16
тип boolean занимает 1 байт 
16 + 1 = 17

Размер объекта всегда должен быть кратен 8, т.к обеспечивает более быстрый доступ к памяти,
а так же упрощает сборку мусора.

тип String не имеет четкой длинны, следовательно точный размер невозможно рассчитать.
Получаем размер User от (17 до 17 + макс.длина String) + выравнивание.

Создадим экземпляр класса User:
User user = new User("Ivan", 25, true);
Получаем размер:
заголовок + int + boolean + String => 12 + 4 + 1 + "Ivan".length * 2 => 25 + выравнивание => 25 + 7 => 32
Примерный размер user = 32 байтам.

Пустой объект, без полей будет занимать 12 байт.

        }
