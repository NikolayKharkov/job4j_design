Создан класс User. Данный класс содержит следующие поля : String name, int age, 
boolean active. Переопрелен метод finalize.
Теоретический расчет:
заголовок объекта занимает 12 байт
0 + 12 = 12
тип int занимает 4 байта
12 + 4 = 16
тип boolean занимает 1 байт 
16 + 1 = 17
тип String не имеет четкой длинны, следовательно точный размер невозможно рассчитать.
Получаем размер User от 17 до 17 + макс.длина String.

Пустой объект, без полей будет занимать 12 байт.

Создал метод main, в котором прописал цикл 
for (int i = 0; i < 10000; i++) {
            new User("N" + i, i, true);
        }
В конфигурации прописал ключи -Xmx10m -Xms10m
Поскольку создаются объекты, но ссылки на них отсутствуют, GC сразу их удаляет.
